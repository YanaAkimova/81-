## 🚀 Как опубликовать

1. Создайте новый репозиторий на GitHub.
2. Загрузите все файлы, сохраняя структуру папок.
3. Перейдите в **Settings → Pages**.
4. В разделе **Source** выберите **Deploy from a branch**.
5. Укажите ветку `main` и папку `/ (root)`.
6. Нажмите **Save** — сайт будет доступен через 1–2 минуты.

## 🖼️ Подготовка фотографий

Фотографии должны лежать в папке `images/` с **точными именами файлов**, указанными выше. Если имя или расширение отличается — фото не отобразится, но сайт покажет подсказку с нужным путём.

Музыкальный файл `den-pobedy.mp3` кладётся в папку `audio/`.
